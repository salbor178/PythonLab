## Descripción  
HuellitasAPP es una aplicación en Python que permite administrar turnos y servicios dentro de una veterinaria.  
El sistema posibilita registrar nuevas atenciones para mascotas, visualizar y cancelar turnos existentes, y generar estadísticas sobre los servicios más solicitados y la recaudación total.
Toda la información se almacena en un archivo de texto, facilitando el control y seguimiento de las actividades del centro veterinario.
